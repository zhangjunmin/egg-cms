## 技术
- 后端：nodejs + express
- 数据库：MongoDB
- 前端：layui，jquery，插件(图片上传,文本编辑器)
- 渲染模板：ejs
## 结构
- app 服务器端文件
  - util.js 工具方法
  - confs 配置文件
  - routers 路由文件
  - 
- public 公共文件
 - vendor 引用的外部资源文件
 - resource 自定义 资源文件
   - js
   - css
   - images
- views 视图文件
- upload 上传图片目录
